# Smart Utility Toolkit

Smart Utility Toolkit is a mobile application built with React Native and Expo.  
It combines useful everyday tools into one clean and easy-to-use app.

## Features

### 1. Unit Converter
Convert values across:
- Length
- Weight
- Temperature
- Currency

### 2. Notes
- Add notes
- Edit notes
- Delete notes
- Persist notes locally using AsyncStorage

### 3. Calculator
- Perform basic arithmetic operations
- Supports addition, subtraction, multiplication, and division
- Includes clear and result actions

## Tech Stack

- React Native
- Expo
- Expo Router
- AsyncStorage
